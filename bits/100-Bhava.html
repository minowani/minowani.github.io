Bhava

Bhava is often translated as becoming, existence.

It is the support for birth. If there is no existence there is no birth, but that is not exactly what it is here about as this is more of an abstract denial; an assumed nothingness. But becoming fits, though it can do with a small explanation.

There are three becomings, which we are already familiar with:

  1. becoming by lust (kāmabhavo)
  2. becoming by form (rūpabhavo)
  3. becoming by 'without form' (arūpabhavo)

Lust is a passional attitude. When not open to lust there yet can be passion for form and for the without form, the perceptions beyond form. And so there there can be higher and lower principles to attend to. Becoming by higher principles is esteemed higher, one might end up in higher places, and if we let ourselves in with lower principles we set things up to end up in lower places.

Becoming by lust doesn't mean that there is then no form, so it involves more of an orientation, direction, grow: If we don't have passion for the same things, we then don't long for the same things, we then don't take the same things on board, we are then not becoming ("growing", "orienting") along the same things, and so we might not end up at same places.

But, thus so:

'With support of taking up, becoming. With support of becoming, birth'.

Pāḷi-English

Bhava becoming.
